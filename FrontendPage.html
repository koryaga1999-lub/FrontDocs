Фронтенд

Что такое Фронтенд?

Фронтенд — это всё, что видит пользователь в браузере: кнопки, меню, анимации, формы. Если сайт — это дом, то фронтенд — его фасад, мебель и интерьер. Фронтенд-разработчик превращает дизайн в рабочий сайт с помощью HTML, CSS и JavaScript.

Из чего состоит Фронтенд?

HTML - каркас (структура) сайта, это заголовки, параграфы, таблицы, формы и т.д.

<button>Кнопка</button>
<h1>Заголовок</h1>

JS (JavaScript) - поведение сайта, интерактив, клики, анимации и т.д

button.addEventListener('click', () => {alert('Привет!');});

CSS - стиль, цвета, размеры, отступы, анимации и т.д.

button { color: white; }
