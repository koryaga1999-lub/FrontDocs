# FrontDocs

# 🚀 FrontDocs — Документация, которая учит. Без воды.

**FrontDocs** — это образовательная платформа и экосистема документации, созданная для тех, кто хочет освоить веб-разработку (Frontend) быстро, структурно и без пустой теории.

Мы верим, что лучший способ научиться программировать — это писать код, сразу понимая логику процессов. Никаких часов переливания из пустого в порожнее. Только суть, практика и чистый код.

## 📖 О проекте

Этот репозиторий содержит исходный код и документацию к сайту [FrontDocs](на хостинг пока не запущен,так как всё ещэ в разработке) .

Основная цель проекта — разрушить стереотип о том, что обучение программированию должно быть скучным и перегруженным лишней информацией. Мы предоставляем:
- **Лаконичные гайды**: Только то, что действительно нужно в реальной разработке.
- **Практические примеры**: Каждый модуль заканчивается работающим кодом.
- **Структуру реального проекта**: Мы учим не просто синтаксису, а архитектуре.

## 🛠 Стек технологий

Сайт построен на современных инструментах, которым мы же и обучаем:

- **Ядро:** HTML5, CSS3, JavaScript (ES6+)
- **Стили:** TailwindCSS / CSS Modules
- **Документация:** MDX / Markdown

## 🗺 Содержание (Дорожная карта)

1.  **Введение в Frontend**
    - Понятие фронтенда.
    - Настройка среды разработки (VS Code, терминал, Git).
2.  **HTML без воды**
    - Семантическая верстка для людей и машин.
    - Доступность (A11y) с первых строк кода.
3.  **CSS для реальных задач**
    - Flexbox и Grid на пальцах.
    - Адаптивность (Mobile First).
4.  **Чистый JavaScript**
    - Асинхронность (Promise, async/await) без боли.
    - Разбор с самого нуля
    - Современный синтаксис (ES2020+).
## 🚀 Быстрый старт

Если вы хотите запустить проект локально, чтобы вносить правки или изучать код, следуйте инструкции:

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/koryaga1999-lub/FrontDocs
git clone git@github.com:koryaga1999-lub/FrontDocs.git # по SSH-Ключу

# 2. Перейдите в папку проекта
cd frontdocs

# 3. Установите зависимости
npm install

# 4. Запустите режим разработки
npm run dev
