Отступ, граница, размер

Вся страница — это коробки в коробках. Каждый элемент HTML — это прямоугольная коробка, которая ведёт себя по своим правилам.

Два типа “коробок”:

Блочные элементы (как большие коробки):

📦 Коробка 1

📦 Коробка 2

📦 Коробка 3

  • Занимают всю ширину
  • Стоят друг под другом
  • Примеры: <div>, <p>, <h1>, <ul>, <ul>, <li>

Строчные элементы (как слова в тексте):

📦 📦 📦 📦 📦 (все в одной строке)

  • Занимают только место под содержимое
  • Стоят рядом в строке
  • Примеры: <span>, <a>, <strong>, <em>

Из чего состоит каждая коробка:

По слоям:

  1. Контент — что внутри (текст, картинка)
  2. Padding — внутренняя подушка (отступ от контента до рамки)
  3. Border — рамка вокруг
  4. Margin — внешний отступ (отделяет от соседних коробок)

Как это работает на практике:

Пример:

                <div>
                <h1>Фронтенд</h1>
                <p>Это <span>замечательная</span> вещь</p>
                </div>

            

                <!-- Большая коробка -->
                <!-- Коробка-заголовок -->
                <!-- р - коробка, span - слово внутри -->

Важные правила

Что можно:

                <!-- Блочный внутри блочного -->
                <div>
                    <p>Текст</p>
                </div>

                <!-- Строчный внутри строчного -->
                <span>Это <strong>важно</strong></span>

                <!-- Строчный внутри блочного -->
                <p>Нажми <a href="#">здесь</a></p>

Что нельзя:

                <!-- Блочный внутри строчного - НЕТ! -->
                <span><div>Текст</div></span>

                <!-- р внутри р - НЕТ! -->
                <p>Текст <p>ещё текст</p></p>

Как проверить в браузере:

  1. Откройте любую страницу
  2. ПКМ → Inspect (Исследовать элемент)
  3. Выберите любой элемент
  4. В панели разработчика увидите:
  • Размеры коробки (ширина и высота)
  • Отступы (padding, margin, border)
  • Тип элемента (блочный или строчный)

Как управлять размерами и отступами:

                div {
                    width: 300px; /* ширина */
                    height: 200px; /* высота */
                    padding: 20px; /* внутренний отступ */
                    border: 2px solid black; /* граница */
                    margin: 10px; /* внешний отступ */
                }
