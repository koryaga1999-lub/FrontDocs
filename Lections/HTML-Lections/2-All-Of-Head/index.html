Всё о head

Если «body» - это содержание страницы, которое видит пользователь, то «head» - это часть страницы, где хранятся метаданные: информация о странице для браузера, поисковые и другие устройства.

1. Что такое <head> и зачем он нужен?

<head> - это служебная часть сайта, которая НЕ видана пользователем.

В ней хранится:

  1. Название страницы (title) - то, что видно во вкладке браузера
  2. Язык сайта (lang) - чтобы браузер знал, на каком языке текст
  3. Кодировка (<meta charset>) - чтобы русские буквы не превращались в "кракозябры"
  4. Настройка для телефонов (<meta name="viewport">) - чтобы сайт нормально выглядел на мобильных устройствах
  5. Описание для Google (<meta name="description">): короткий текст, который показывает в поиске
  6. Ссылки на стили - откуда брать цвета, шрифты и оформление
  7. Иконка сайта (favicon): маленькая картинка во вкладке браузера

2. <title> - название страницы

Название страницы видно во вкладках браузера, закладках и в поиске (Google / Yandex):

<title>Рецепт борща</title>

3. Кодировка: <meta charset>

Кодировка в head говорит браузеру: "используй русские буквы, иероглифы, эмодзи - всё что угодно"

<meta charset="utf-8">

Пишется самым первым в head.

Если же забыть написать кодировку, то вместо понятных нам всем символов мы получим примерно вот это: РєР°РєРѕР№-С‚Рѕ РЅРµРїРѕРЅСЏС‚РЅС‹Р№ С‚РµРєСЃС‚

4. Метаданные для поисковиков: <meta name="...">

Элемент <meta> с атрибутами name и content добавляет информацию о странице, полезную для поисковиков и разработчиков. Два популярных типа:

Автор (author):

Указывает, кто создал страницу.

<meta name="author" content="Иван Иванов">

Это помогает идентифицировать автора.

Описание (description):

Краткое описание страницы для поисковиков (SEO).

<meta name="description" content="Страница о книге Автостопом по галактике: приключения, фантастика и поиск Великого ответа">

5. Подключение иконок: favicon и другие

Что это?

Маленькая картинка рядом с названием сайта.

Где видна:

  1. Во вкладке браузера
  2. Закладки
  3. В истории посещений
  4. На телефоне (если добавить сайт на главный экран)
<link rel="icon" href="images/favicon.ico">

Иконки для устройств (например, iPhone, iPad):

Используются для экрана с разными размерами.

<link rel="apple-touch-icon" sizes="144x144" href="icon144.png">
<link rel="apple-touch-icon" sizes="72x72" href="icon72.png">

6. Подключение CSS и JavaScript

CSS подключается с помощью link в head:

<link rel="stylesheet" href="style.css">

JavaScript с помощью <script> и <noscript> в конце body:

<script src="script.js"></script>

Правильный порядок:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Название страницы</title>
    <meta name="description" content="Описание страницы">
    <link rel="icon" href="favicon.ico">
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <!-- Содержимое страницы -->
    <script src="script.js"></script>
</body>
</html>

7. Установка языка страницы

Чтобы установить язык страницы нужно написать:

<html lang="ru">

Это помогает:

  • Поисковикам правильно индексировать страницу.
  • Программам для слабовидящих (скринридерам) читать текст с правильным произношением.
