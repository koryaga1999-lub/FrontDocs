Всё о head

Если <body> - это содержимое страницы, которое видят пользователи, то <head> - это «мозг» страницы, где хранятся метаданные: информация о странице для браузеров, поисковиков и других устройств.

Что такое <head> и зачем он нужен?

<head> - это служебная часть сайта, которая НЕ видна пользователям. В ней хранится:

  1. Название страницы (title) - то, что видно во вкладке браузера
  2. Язык сайта (lang) - чтобы браузер знал, на каком языке текст
  3. Кодировка (<meta charset>) - чтобы русские буквы не превращались в "кракозябры"
  4. Настройки для телефонов (<meta name="viewport">) - чтобы сайт нормально выглядел на мобильных устройствах
  5. Описание для Google (<meta name>) - короткий текст, который показывают в поиске
  6. Ссылки на стили  - откуда брать цвета, шрифты и оформление
  7. Иконка сайта (favicon) - маленькая картинка во вкладке браузера

<tittle> - название страницы

Название страницы видно во вкладках браузера, закладках и в поиске (Google / Yandex):

<title>Рецепт борща</title>

<tittle> отличается от <h1> тем, что h1 это заголовок на самой странице, а tittle - название вкладки.

Кодировка: <meta charset>

Кодировка в head говорит браузеру: "используй русские буквы, иероглифы, эмодзи - всё что угодно"

<meta charset="utf-8">

Пишется самым первым в head.

Метаданные для поисковиков: <meta name>

Элемент <meta> с атрибутами name и content добавляет информацию о странице...

  1. Автор (author): указывает, кто создал страницу.

    <meta name="author" content="Иван Иванов">

    Это помогает идентифицировать автора

  2. Описание (description): краткое описание страницы для поисковиков (SEO).

    <meta name="description" content="Страница о книге...">

Подключение иконок: favicon и другие

Что это?

Маленькая картинка рядом с названием сайта:

[🔖] Название сайта <- эта иконка

Где видна:

  1. Во вкладке браузера 
  2. В закладках 
  3. В истории посещений 
  4. На телефоне (если добавить сайт на главный экран)
<link rel="icon" href="images/favicon.ico">

Иконки для устройств (например, iPhone, iPad):Используются для экрана с разными размерами.

<link rel="apple-touch-icon" sizes="144x144" href="favicon144.png">
<link rel="apple-touch-icon" sizes="72x72" href="favicon72.png">

Подключение CSS и JavaScript

CSS подключается с помощью link в head:

<link rel="stylesheet" href="style.css">

JavaScript с помощью <script></script>в конце body

<script src="script.js"></script>

Правильный порядок:

                        <!DOCTYPE html>
                        <html>
                        <head>
                        <!-- 1. Кодировка -->
                        <meta charset="utf-8">

                        <!-- 2. CSS туг -->
                        <link rel="stylesheet" href="style.css">
                        </head>
                        <body>
                        <!-- 3. Весь HTML -->

                        <!-- 4. JS в конце -->
                        <script src="script.js"></script>
                        </body>
                        </html>

Установка языка страницы

Чтобы установить язык странице нужно написать:

<html lang="ru">

Это помогает:

  • Поисковикам правильно индексировать страницу.
  • Программам для слабовидящих (скринридерам) читать текст с правильным произношением.
