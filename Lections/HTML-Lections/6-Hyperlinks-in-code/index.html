Создание гиперссылок в коде

Что такое гиперссылка?

Это кликабельный элемент, который ведёт куда-то: на другую страницу, файл, почту или внутри текущей страницы. Без ссылок Интернет был бы просто набором отдельных страниц - как дома без дорог между ними.

Как создать ссылку?

Базовая формула:

<a href="ссылка">кликабельный текст/a>

Примеры:

                        <a href="https://twitch.tv">Перейти на Твич</a>
                        <a href="page2.html">На вторую страницу</a>
                        <a href="#section2">К разделу 2</a>

Главные атрибуты тега <a>:

  1. href — куда идём (обязательный для рабочей ссылки) 
    <a href="https://yandex.ru">Поиск в Яндексе</a>
  2. target — где открываем (чаще всего _blank для внешних ссылок) 
    <a href="https://google.com" target="_blank">Google в новой вкладке</a>
  3. title — всплывающая подсказка (для дополнительной информации) 
    <a href="/about.html" title="Информация о нашей компании">О нас</a>
  4. download — скачать, а не открыть (для файлов) 
    <a href="/files/document.pdf" download="Руководство.pdf">Скачать инструкцию</a>
  5. id — уникальное имя (для якорных ссылок внутри страницы) 
    <a href="#contacts" id="contact-link">К контактам</a>

Два типа адресов:

  • Абсолютный - полный адрес: https://site.com/page.html(для ссылок на другие сайты)
  • Относительный - путь от текущей страницы: page.html или folder/page.html(для ссылок внутри своего сайта)

Правило: внутри своего сайта - относительные, на чужие сайты - абсолютные.

Три особых типа ссылок:

  1. Закладки (якоря) — прыжок внутри страницы: 
                                
                                <!-- 1. Ставим якорь -->
                                <h2 id="contacts">Контакты</h2>

                                <!-- 2. Ссылаемся на него -->
                                <a href="#contacts">К контактам</a>
  2. Ссылка на почту: 
    <a href="mailto:hello@site.com">Написать нам</a>
  3. Ссылка для скачивания: 
    <a href="file.pdf" download>Скачать PDF</a>

Запомните!

  1. Без href ссылка не работает - это просто текст
  2. Одна страница - один id - не повторяйте одинаковые якоря
  3. Ссылка должна быть понятна даже без контекста
  4. Проверяйте все ссылки - битые ссылки раздражают пользователей
