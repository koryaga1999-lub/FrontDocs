Создание видео и аудио в коде

Видео: тег <video>

                    <video controls width="600">
                        <source src="video.mp4" type="video/mp4">
                        <source src="video.webm" type="video/webm">
                        Ваш браузер не поддерживает видео
                    </video>

Аудио: тег <audio>

                    <audio controls>
                        <source src="audio.mp3" type="audio/mpeg">
                        <source src="audio.ogg" type="audio/ogg">
                        Ваш браузер не поддерживает аудио
                    </audio>

Обязательные атрибуты:

controls - добавляет кнопки управления (воспроизведение, громкость)

<source> внутри - указывает разные форматы файлов для совместимости

Дополнительные атрибуты:

  1. autoplay - начать автоматически (осторожно! Раздражает пользователей)
  2. loop - повторять бесконечно
  3. muted - без звука по умолчанию
  4. poster (только для видео) - картинка-превью перед воспроизведением
  5. width/height (только для видео) - размеры плеера

Форматы файлов - что использовать:

Видео:

  • MP4 (video/mp4) - работает везде (Safari, Chrome, Firefox)
  • WebM (video/webm) - современный, меньше размер
  • Всегда указывайте оба через 

Аудио:

  • MP3 (audio/mpeg) - работает везде
  • OGG (audio/ogg) - альтернатива для Firefox/Chrome
  • WAV (audio/wav) - без сжатия, большой размер

Доступность: субтитры <track>

                    <video controls>
                        <source src="video.mp4" type="video/mp4">
                        <track kind="subtitles" src="subtitles.vtt" srclang="ru" label="Русский">
                    </video>
  • kind="subtitles" - перевод диалогов
  • kind="captions" - описание звуков для слабослышащих
  • Формат WebVTT (файл .vtt) с таймингом
  • Улучшает SEO - поисковики читают текст

Правила для новичков:

  1. Всегда добавляйте controls - без них пользователь не сможет управлять
  2. Не используйте autoplay с громким звуком - это раздражает
  3. Указывайте несколько форматов через <source> для совместимости
  4. Добавляйте резервный текст между тегами для старых браузеров
  5. Сжимайте файлы - видео должны быть оптимизированы
  6. Для субтитров создавайте файлы .vtt с таймингом

Быстрый пример работающего видео:

                    <video controls width="100%" poster="preview.jpg">
                        <source src="travel.mp4" type="video/mp4">
                        <source src="travel.webm" type="video/webm">
                        <track kind="subtitles" src="subtitles.vtt" srclang="ru">
                        <p>Скачайте <a href="travel.mp4">видео о путешествиях</a></p>
                    </video>

Чего не нужно делать:

  1. Не использовать один формат
  2. Не забывать про controls
  3. Не включать громкий autoplay
  4. Не загружать гигантские файлы без сжатия
  5. Не игнорировать субтитры

Запомните: Видео и аудио делают сайт живым, но должны быть удобными для всех пользователей — с кнопками управления, разными форматами и субтитрами.
