Итоговое задание по HTML

Творческое задание (ТЗ):

Задание должно быть выполнено так же как и на скриншоте:

<p>Создай одностраничный сайт index.html на тему "Моё хобби" с обязательными элементами: заголовок <h1>, одна фотография <img>, 2-3 предложения текста <p>, список из 3-4 пунктов <ul>, таблица 3×3 <table>, форма с 2 полями и кнопкой <form>, используя минимальный CSS для читаемости который уже для тебя написан — всё в одном файле, поставь себе время выполнения 30-45 минут, если получится дольше, не переживай, это нормально, поскольку ты ещё учишься!</p>
