Формы - взаимодействие с пользователем

Базовая структура формы:

<form action="/обработчик" method="POST">

    <!-- Поля формы -->

    <button type="submit">Отправить</button>

</form>

  • action - куда отправлять данные
  • method - способ отправки (POST или GET)
  • POST - для паролей, личных данных
  • GET - для поиска, фильтров (данные видны в URL)

Все важные атрибуты, использующиеся в формах:

name — имя для данных при отправке
id — уникальный ярлык элемента
value — предзаполненный текст в поле
placeholder — серая подсказка в пустом поле
required — обязательно к заполнению
type — вид поля (текст, пароль, email)
for — связь label с полем по id
action — куда отправлять форму
method — способ отправки (GET/POST)
disabled — поле отключено
readonly — только чтение
maxlength — лимит символов
min/max — минимальное/максимальное значение
pattern — шаблон для проверки текста
autocomplete — всплывающие подсказки
autofocus — автофокус при загрузке
multiple — можно выбрать несколько вариантов
rows/cols — размеры текстового поля
selected — выбранный пункт по умолчанию
checked — отмеченная галочка/кнопка

Основные элементы формы:

<input type="text" placeholder="Ваше имя" required>

<input type="email" placeholder="Email">

<input type="password" placeholder="Пароль">

  • required - делает обязательное поле для заполнения
  • placeholder - подсказка внутри заполняемого поля

Чекбоксы и радиокнопки:

Радиокнопки (только один выбор)

<input type="radio" name="gender" value="male" id="male">

<label for="male">Мужской</label>

Чекбоксы (несколько выборов)

<input type="checkbox" name="interests" value="sports" id="sports">

<label for="sports">Спорт</label>

Выпадающий список:

<select name="city">

  <option value="">Выберите город</option>

  <option value="moscow">Москва</option>

  <option value="spb">Санкт-Петербург</option>

</select>

Многострочный ввод:

<textarea rows="4" cols="50" placeholder="Ваш комментарий">

Доступность форм - обязательно:

Связываем label с input:

<label for="username">Имя пользователя: </label>

<input type="text" id="username" name="username">

  • for в label должен совпадать с id в input
  • Увеличивает область клика - можно кликнуть на текст label
  • Необходимо для скринридеров

Группировка полей:

<fieldset>

  <legend>Контактная информация</legend>

  <-- поля формы -->

</fieldset>

Типы кнопок:

<button type="submit">Отправить </button> - отправка формы

<button type="reset">Очистить </button> - сброс полей

<button type="button">Просто кнопка </button> - для JS

Валидация (проверка данных):

Браузерная валидация:

<button type="submit">Отправить /button - отправка формы>

<button type="reset">Очистить /button - сброс полей>

<button type="button">Просто кнопка /button - для JS>

Пример полностью рабочей формы:

<form action="/subscribe" method="POST">
    <fieldset>
        <legend>Выбор темы новостей с рассылкой</legend>
        
        <label for="name">Имя:</label>
        <input type="text" id="name" name="name" required>
        
        <label for="email">Email:</label>
        <input type="email" id="email" name="email" required>
        
        <select id="news" name="news">
            <option value="cooking">Кулинария</option>
            <option value="travel">Путешествия</option>
        </select>
        
        <button type="submit">Подписаться</button>
    </fieldset>
</form>

CSS для стилизации форм:


            /* Базовая стилизация */
            input, select, textarea {
                width: 100%;
                padding: 10px;
                margin: 5px 0;
                border: 1px solid #ddd;
                border-radius: 4px;
            }

            /* Стиль при фокусе */
            input:focus, textarea:focus {
                border-color: #4a6fa5;
                outline: none;
                box-shadow: 0 0 5px rgba(74, 111, 165, 0.3);
            }

            /* Стиль для невалидных полей */
            input:invalid, textarea:invalid {
                border-color: #e74c3c;
            }

            /* Стиль кнопок */
            button {
                background-color: #4a6fa5;
                color: white;
                padding: 10px 20px;
                border: none;
                border-radius: 4px;
                cursor: pointer;
            }

            button:hover {
                background-color: #3a5a80;
            }

Главные правила для новичков:

Всегда:

  1. Используйте <label> для каждого поля
  2. Добавляйте id для полей и связывайте с for в label
  3. Используйте <fieldset> и <legend> для группировки
  4. Проверяйте валидацию (required, type="email")
  5. Тестируйте клавиатурную навигацию (Tab, Enter)

Никогда:

  1. Не создавайте формы без label
  2. Не используйте таблицы для вёрстки форм
  3. Не полагайтесь только на браузерную валидацию
  4. Не забывайте про мобильные устройства
  5. Не делайте слишком длинные формы

Вопросы для самопроверки вашей формы:

  1. Есть ли у всех полей <label>?
  2. Совпадают ли id и for?
  3. Работает ли навигация с Tab?
  4. Понятны ли сообщения об ошибках?
  5. Адаптирована ли форма для мобильных?

Запомните: У каждого тега своя задача. Правильно подобранные теги делают форму понятной для пользователей, скринридеров и поисковиков!
