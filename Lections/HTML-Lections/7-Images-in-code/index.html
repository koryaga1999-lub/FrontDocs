Создание изображения в коде

Как добавить картинку на страницу

<img src="путь/к/картинке.jpg" alt="Описание картинки">

  • src - откуда брать картинку (путь или URL)
  • alt - что это за картинка (обязательно!)

Обязательный атрибут alt

Это не опция, а необходимость:

Так делать можно:

<img src="logo.png" alt="Логотип компании Яндекс">

<img src="poster.jpg" alt="Афиша кинотеатра Октябрь">

<img src="decor.png" alt=""> <!-- пустой для декоративных картинок -->

Так делать не нужно:

<img src="poster.jpg" alt="картинка"> <!-- слишком общее -->

<img src="poster.jpg" alt=""> <!-- не для информативных картинок -->

<img src="poster.jpg"> <!-- БЕЗ alt - ОШИБКА -->

Правила для alt:

  1. Для информативных картинок - опишите что на картинке
  2. Для декоративных - пустой alt=""
  3. Кратко и по делу - не больше 1-2 предложений
  4. Не начинайте с "Картинка..." - скринридер и так знает что это картинка

Размеры картинок

<img src="photo.jpg" alt="Фото" width="600" height="400">

  • Ширина и высота помогают браузеру заранее выделить место
  • Не искажайте - используйте оригинальные пропорции
  • Для адаптивности лучше использовать CSS, о котором мы ещё поговорим в дальнейших лекциях

<figure> и <figcaption> - для картинок с подписями

<figure>

<img src="poster.jpg" alt="Афиша спектакля">

<figcaption>Спектакль "Гамлет" - 15 мая 2025</figcaption>

</figure>

  • <figure> - контейнер для медиаконтента
  • <figcaption> - подпись к картинке

Используйте для диаграмм, схем, иллюстраций и т.п.

Какие форматы изображений использовать?

JPEG - фото, без прозрачности, сжимается с потерями

PNG - лого, с прозрачностью, без потерь качества

WebP - современно, меньше вес, с прозрачностью

SVG - иконки, вектор, масштабируется без потерь

GIF - простые анимации, 256 цветов

AVIF - самое лучшее сжатие, но поддержка не везде

Картинки в CSS vs HTML

.hero {background-image: url('bg-pattern.png');}

CSS

<img src="content-image.jpg" alt="Важная информация">

HTML

Правило:

  • HTML-изображения - для контента (видны всем и индексируются)
  • CSS-изображения - для украшений (не видны поисковикам)

Важные правила для новичков

  1. Всегда добавляйте alt - даже если картинка декоративная (alt="")
  2. Проверяйте пути - src="images/photo.jpg" а не src="photo.jpg"
  3. Сжимайте картинки - используйте TinyPNG или Squoosh
  4. Давайте понятные имена файлов:
  • hamlet-poster-2025.jpg ✅
  • IMG_12345.jpg ❌
