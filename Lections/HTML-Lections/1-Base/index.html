Основы работы с HTML

HTML - это язык разметки, с помощью которого создаётся структура веб-страницы.

Простыми словами:

HTML говорит браузеру: «вот здесь будет заголовок», «здесь - картинка», «тут - кнопка», «а здесь - список».

Например:

<h1>Мой первый сайт</h1> → Это заголовок (h = head)
<p>Привет, мир!</p> → Это абзац текста (p = paragraph)
<img src="cat.jpg"> → Это картинка котика
<a href="https://ya.ru">Ссылка на Яндекс</a> → Это ссылка

Браузер читает этот код и рисует страницу с заголовком, текстом, картинкой и ссылкой.

Что такое теги?

В HTML эти угловые скобки (<h1>, <p>, <img>) называются тегами. Теги оборачивают наш контент (текст, картинки, ссылки), чтобы браузер понимал, что это такое.

Например, тег <section> создаёт секцию на сайте, внутри которой может быть текст, картинки и другие элементы.

Каждый тег можно использовать бесконечное количество раз и много раз вкладывать их в секцию.

Иногда вложенность - главная часть разметки, например при построении таблиц, о которых Вы узнаете позже.

Вот к примеру список:

<ul> - объявление списка
<li>Первый элемент списка</li>
<li>Второй элемент списка</li>
<li>Третий элемент списка</li>
</ul>

Браузер выведет эту HTML-разметку следующим образом:

  • Первый элемент списка
  • Второй элемент списка
  • Третий элемент списка

Без тегов <li> браузер не сможет отличить простой текст от элементов списка.

Атрибуты

Атрибуты - это просто дополнительная информация для браузера.

В HTML вы можете использовать ссылки, чтобы перемещаться по интернету.
Недостаточно просто указать тег ссылки <a></a> и текст внутри.

<a>Текст ссылки</a> - ничего не произойдёт и Вас к сожалению никуда не перенаправят самостоятельно :(

Чтобы перенаправить пользователя по ссылке, к тегу <a> нужен специальный атрибут href, значением которого является нужная Вам ссылка:

<a href="https://twitch.tv">Пойти смотреть стримы!</a>

Теперь, при клике на текст (указанный в атрибуте) Вы будете перенаправлены на ссылку, которую указали.
